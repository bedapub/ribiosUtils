# Whether an integer is odd (or even)

Whether an integer is odd (or even)

## Usage

``` r
isOdd(x)

isEven(x)
```

## Arguments

- x:

  An integer.

## Value

Logical, whether the input number is odd or even.

`isOdd` and `isEven` returns whether an integer is odd or even,
respectively.

## Examples

``` r
isOdd(3)
#> [1] TRUE
isEven(4)
#> [1] TRUE
```
