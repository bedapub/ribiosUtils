# Translate well index numbers to well positions

Translate well index numbers to well positions

## Usage

``` r
wellIndex2position(ind, format = c("96", "384"))
```

## Arguments

- ind:

  Well index, integer numbers starting from 1, running rowwise.
  Non-integer parameters will be coereced to integers.

- format:

  Character string, well format

## Value

A `data.frame` containing three columns: input `WellIndex`, `Row`
(characters) and `Column` (integers)
