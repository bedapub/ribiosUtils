# Print BEDA project information

Print BEDA project information

## Usage

``` r
bedaInfo()
```

## Value

A list, including pstore path, URL, git address, and user id The
function is used at the end of the Rmarkdown report to print relevant
information to help other colleagues finding relevant resources

## Examples

``` r
bedaInfo()
```
