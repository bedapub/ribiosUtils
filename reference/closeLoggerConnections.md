Close connections to all loggers This function closes all open
connections set up by loggers It is automatically run at the end of the
R session (setup by
[`registerLog`](https://bedapub.github.io/ribiosUtils/reference/registerLog.md))

## Usage

``` r
closeLoggerConnections()
```

## Value

Invisible `NULL`. Only side effect is used.
