# ribiosUtils: Utilities from and Interface to the 'Bioinfo-C' ('BIOS') Library

Provides interface to the 'Bioinfo-C' (internal name: 'BIOS') library
and utilities. 'ribiosUtils' is a Swiss-knife for computational biology
in drug discovery, providing functions and utilities with minimal
external dependency and maximal efficiency.
